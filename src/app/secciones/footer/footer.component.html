A cerca de VMI

VMI es una apicación diseñada para que municipales puedan mantener un control vial y se hagan cumplir las leyes de transito, pudiendo participar policias municipales como transeúntes. Esta idea fue pensada en el año 2020 para evitar y reducir infracciones viales que se producen a diario en la ciudad de Villa María, Córdoba.

Herramientas Utilizadas
  • Angular 9
  • Node JS
  • FireBase
  • Bootstrap v4.5
  • Visual Studio Code
  • Angular Material
  • Code Pen
  • Pexels
